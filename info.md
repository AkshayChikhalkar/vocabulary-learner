# Vocabulary Learner

A comprehensive vocabulary learning integration for Home Assistant with spaced repetition, progress tracking, and smart notifications.

## Key Features

- **Smart Learning**: Uses SM-2 spaced repetition algorithm for optimal retention
- **Flexible Input**: Support for CSV, JSON, TXT, and TSV file formats
- **Progress Tracking**: Detailed statistics on your learning journey
- **Smart Notifications**: Configurable notifications with quiet hours
- **API Fallback**: Automatic word lookup via Wiktionary and LibreTranslate
- **Multi-language**: Learn any language with full Unicode support

## How It Works

1. **Load Vocabulary**: Import words from a file or use API fallback
2. **Spaced Repetition**: Words are scheduled for review based on difficulty
3. **Track Progress**: Mark words as known/unknown to update learning algorithm
4. **Get Notifications**: Receive periodic reminders to review vocabulary
5. **Monitor Stats**: Track your progress, streaks, and learning statistics

## Use Cases

- Learn a new language vocabulary
- Study technical terms
- Memorize definitions
- Practice spelling and pronunciation
- Build vocabulary for exams

## Getting Started

1. Install the integration via HACS or manually
2. Add your vocabulary file to the config directory (optional)
3. Configure the integration with your preferences
4. Start learning!
