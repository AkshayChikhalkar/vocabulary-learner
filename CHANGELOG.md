# Changelog

All notable changes to the Vocabulary Learner integration will be documented in this file.

## [1.0.0] - 2024-01-01

### Added
- Initial release of Vocabulary Learner integration
- Support for multiple vocabulary file formats (CSV, JSON, TXT, TSV)
- SM-2 spaced repetition algorithm for optimal learning
- Progress tracking (known/unknown/learning words)
- Configurable notifications with quiet hours
- API integration with Wiktionary and LibreTranslate
- Statistics tracking (progress, streaks, words per day)
- Services for marking words, getting next word, resetting progress, and exporting data
- Multi-language support
- Comprehensive documentation

### Features
- File-based vocabulary loading
- Automatic format detection
- Spaced repetition scheduling
- Notification system
- Progress persistence
- Export/import functionality
